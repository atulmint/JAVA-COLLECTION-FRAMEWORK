## üìò **1. What Is a Stack?**

* A **Stack** is a **Last-In, First-Out (LIFO)** data structure.
* The element added **last** is the **first** to be removed.
* It is part of the `java.util` package and extends the `Vector` class.
* Common operations:

  * **push()** ‚Üí Add an element
  * **pop()** ‚Üí Remove the top element
  * **peek()** ‚Üí View the top element
  * **empty()** ‚Üí Check if the stack is empty

---

## üß† **2. Example Code**

```java
import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        Stack<String> animals = new Stack<>();

        animals.push("Lion");
        animals.push("Dog");
        animals.push("Horse");
        animals.push("Cat");

        System.out.println("Stack: " + animals);

        System.out.println("Top element (peek): " + animals.peek());

        animals.pop();  // Removes the top element ("Cat")

        System.out.println("Stack after pop: " + animals);

        System.out.println("Top element now (peek): " + animals.peek());
    }
}
```

## üßæ **3. Step-by-Step Explanation**

| Operation               | Description                                | Output                    |
| ----------------------- | ------------------------------------------ | ------------------------- |
| `animals.push("Lion")`  | Adds ‚ÄúLion‚Äù to the stack                   | `[Lion]`                  |
| `animals.push("Dog")`   | Adds ‚ÄúDog‚Äù                                 | `[Lion, Dog]`             |
| `animals.push("Horse")` | Adds ‚ÄúHorse‚Äù                               | `[Lion, Dog, Horse]`      |
| `animals.push("Cat")`   | Adds ‚ÄúCat‚Äù                                 | `[Lion, Dog, Horse, Cat]` |
| `animals.peek()`        | Shows top element (`Cat`) without removing | `Cat`                     |
| `animals.pop()`         | Removes top element (`Cat`)                | `[Lion, Dog, Horse]`      |
| `animals.peek()`        | Now shows new top (`Horse`)                | `Horse`                   |

## üí° **5. Key Points to Remember**

* Stack follows **LIFO (Last In, First Out)** principle.
* `push()` ‚Üí Add
* `pop()` ‚Üí Remove top element
* `peek()` ‚Üí View top element without removing it
* `empty()` ‚Üí Returns `true` if stack has no elements
